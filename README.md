# AeroMind Nexus - Unified Aviation Disruption Intelligence

> **Predict. Simulate. Prevent global aviation disruption.**

## 🚀 Quick Start

```bash
# Install dependencies
npm install

# Set up environment variables (create .env file)
DATABASE_URL=postgresql://user:password@localhost:5432/aeromind
SESSION_SECRET=your-secret-key

# Initialize database
npm run db:push

# Start development server
npm run dev
```

Visit: **http://localhost:5000**

## ✨ Features

- 🔮 **Cascading Disruption Prediction** - Graph Neural Networks predict delay propagation
- 🎯 **What-If Simulation Engine** - Monte Carlo optimization for decision-making
- 👁️ **AI Damage Detection** - Computer vision for aircraft inspections
- 📊 **Real-time Dashboard** - Live network visualization and analytics
- 🌐 **6 Complete Pages** - Home, Platform, Technology, Customers, Vision, Demo

## 🛠️ Tech Stack

**Frontend:** React 18, TypeScript, Vite, TailwindCSS, Framer Motion, Recharts
**Backend:** Express.js, PostgreSQL, Drizzle ORM
**AI/ML:** Simulated services (Graph Neural Networks, Computer Vision, Optimization)
